Projet TER

Comment repérer le mauvais comportement des utilisateurs d'une plate-forme web grâce aux traces d'exécution.

Ce projet a débuté lors des Travaux Encadrés de Recherches et s'est poursuivi en stage personnel. Le but est de créés une base de connaissances de l'expérience utilisateur d'une plate-forme web à partir de la lecture des traces d'exécutions. La principale problématique a été la sélection des informations et comment construire un graphe représentatif de ces données et comment faire émerger de la connaissance à partir de ce dernier.

Les données ont été fournis à partir de la plate-forme web SEDIT de l'entreprise Berger-Levrault.

  • Date: Avril - Juillet 2014
  • Catégorie: Base de données Graphe - Réseaux lexicale

Tours de Hanoii

Développement C# XNA - A*

Ce projet a été réalisé en Jeux Vidéo lors de ma dernière année de licence. Le but de ce projet est de comprendre la notion de boucle de jeux dans un modèle MVC. En effet il possible de passer d'une vue 2D à une vue 3D. Le modèle de jeu se base sur l'algorithme A* qui permet d'explorer les possibilités de changement à un instant t grâce à une heuristique.

Vous pouvez télécharger les sources du projet.
